오늘의 과제
"숫자 맞추기"

1. 단판 승부(원하신다면 반복문 사용 가능)
2. 게임이 시작되면, 컴퓨터가 랜덤으로 숫자를 하나 변수에 할당합니다
(범위 : 0 ~ 36) 생성된 숫자는 처음에는 표시하지 않습니다
3. 플레이어는 다른 변수의 입력을 해서 컴퓨터가 생성한 숫자가 무엇인지 맞춥니다
4. 플레이어는 추가로 다른 변수에 입력을 하나 더 해서, 자신의 선택에 옵션을 줍니다
(옵션은 다음과 같습니다)
옵션1 : 내가 선택한 그 숫자가 완전히 맞다
옵션2 : 오차가 1 있을 것 같다
옵션3 : 오차가 2 있을 것 같다
옵션4 : 내 선택이 홀수면 어쨌든 맞추는 숫자도 같은 홀수 같고, 짝수면 같은 짝수 같다(이것이 하나의 선택지)
옵션5 : 어쨌든 맞추는 숫자가 내 결정보다 크거나 같은 것 같다
옵션6 : 어쨌든 맞추는 숫자가 내 결정보다 작거나 같은 것 같다
5. 컴퓨터는 플레이어가 제시한 숫자와, 옵션을 각각 정용해서 다음 점수를 부여합니다(플레이어 변수)
옵션1 : 칼같이 맞추면 35점
옵션2 : 오차 1 있을 것 같다고 하고 맞추면 25점(이 경우, 칼같이 맞춰도 25점)
옵션3 : 오차 2라고 하고 맞추면 15점(칼같이 맞춰도 15점)
옵션4 : 홀짝을 맞추면 10점
옵션5 : 크기를 크거나 작거나 등 맞추면 5점
6. 선택이 끝난 후, 컴퓨터의 랜덤 숫자와, 플레이어의 선택, 선택의 따른 점수를 표시
7. 와일드 카드 : 컴퓨터가 0을 선택했을 경우, 플레이어가 0을 딱 선택하지 않았으면 무조건 틀린 것
