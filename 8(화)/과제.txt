오늘의 과제 : 싱글턴의 복습 체험

0. 오늘 강의에서 쓴 싱글턴은 싱글턴 안에서 데이터가 존재했습니다
1. 이걸 나눠서, 데이터가 있는 데이터 클래스와, 싱글턴 클래스를 따로 만들어봅시다
2. 데이터 클래스 : 캐릭터를 표현한 스테이터스 / 카드 게임 / 사칙 연산도 괜찮습니다
                       클래스 안에 변수가 있고, 변수를 출력 / 접근할 public 함수가 있으면 됩니다
3. 싱글턴 클래스 : 이 안에 데이터 클래스를 변수로 만들어주세요
4. main에서 싱글턴 클래스를 통해서 데이터 클래스를 초기화하고, 연산하고 출력해주세요

예시 : 싱글턴->다른클래스->더하기(1, 2);
