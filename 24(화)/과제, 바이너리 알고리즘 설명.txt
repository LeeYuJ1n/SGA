25일 예습 : 바이너리 리더와 라이터 클래스 구현

[ 설명 ] 바이너리 리더, 라이터 : 바이너리 알고리즘의 응용 중 하나, 컴퓨터 파일을 이진 방식으로 읽는 것
기초적인 파일 입출력이라서 자주 쓰이진 않지만, 적소에 쓰이면 대단히 좋은 효율을 내기도 한다
( * 바이너리 리더, 라이터의 경우 우리가 알고리즘 그 자체를 만들 것이 아니다 : Direct X, Windows의 기능을 써서 실제 파일의 내용을 읽고 쓰겠다는 것)

바이너리 : 2진수, 0 / 1, 딱 둘
바이너리 알고리즘 : 어떤 조건을 제시한 후 딱 그 조건이 true / false인지만 가지고 연산 수행 -> if (true)

<사용>
(1) 파일 입출력 : 파일을 무조건 0과 1로 재구성 
	A. 검색, 구분 : 목록을 딱 반으로 나눠가면서 연산을 수행
	B. 의사 결정 : 대원칙에 현 상황이 맞느냐 아니냐
