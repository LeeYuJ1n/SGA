주말의 과제

렌더 타겟과 뎁스-스텐실 클래스를 응용하면서 다음 구현에 도전해봅시다

[ 쉬움 ]
1. 캐릭터 상태창 비슷한 거 만들기
	└ 휴먼(혹은 나루토 혹은 다른 모델)을 월드에 띄워주세요
	└ 같은 모델을 다른 곳에 하나 더 띄워주세요
	└ 해당 모델을 월드가 아닌 타겟에 투사해주세요
	└ 투사된 모델이 있는 쿼드를 조작키에 따라 ( 예 : I키 ) 켜고 꺼지게 해주세요
	└ 가능하면 쿼드 주변에 사각형 테두리 텍스처도 있으면 좋을 것 같습니다
	└ 그렇게 월드 모델과 렌더 타겟 전용 모델을 각각 월드 / 렌더타겟에 띄워서 캐릭터 상태창을 보는 것 같은 느낌을 만들어주세요
	   ( * 실제로 이 과정이 세련되게 다듬어지면 그게 캐릭터 상태창이 맞습니다 )

[ 어려움 ]
1. 거울 만들기
	└ 월드에 모델을 구현해주세요
	└ 월드에 쿼드를 구현해주세요 (혹은 얇은 큐브도 좋습니다만 쿼드가 제일 좋습니다)
	└ 쿼드 너머에 같은 모델을 하나 더 구현해주세요 (인스턴싱이면 더 좋습니다)
	└ 해당 쿼드가 월드에 있는 모델을 렌더타겟 하듯이 잘라서 투사하게 해주세요
	   ( 스텐실 버퍼를 사용한 3D 거울의 원리 = 해당 영역을 통과한 모델만 렌더하는 것! )

* 어려움 과제는 거의 확실하게 강의 중 작성했던 클래스 중 일부( 특히 뎁스 스텐실, 렌더 타겟)를 수정해야 할 겁니다
